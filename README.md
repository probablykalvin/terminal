# Terminal Style Portfolio

A minimalist personal portfolio page designed to look like a terminal interface. Based on [terminal-style-portfolio-page](https://github.com/heberleonard2/terminal-style-portfolio-page).

## Commands

- `about me` — Bio and background
- `stuff` — Accomplishments and experience
- `skills` — Skills and strengths
- `ping` — Ping the server
- `projects` — GitHub profile
- `social -a` — Social links
- `help` — List all commands
- `clear` — Clear the terminal

## Run locally

Open `index.html` in your browser, or serve the folder with any static file server.

## Deploy

Host on GitHub Pages, Netlify, or any static hosting by uploading these files.
